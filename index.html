The Basic Language of the Web: HTML

Posted by Laura Jones on Monday, June 21st 2027

All modern websites and web applications are built using three fundamental technologies: HTML, CSS and JavaScript. These are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you should learn it.

What is HTML ?

HTML stands for HyperTextMarkup Language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different types of content: paragraphs, links, images, video,etc. Web browsers understand HTML and render HTML code as websites

In HTML, each element is made up of 3 parts

  1. The opening Tag
  2. The closing Tag
  3. The actual element

You can learn more at the MDN Web Docs.
